# home-leis-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态形式存放，不绑定任何特定域名或网站，仅作为信息展示和页面归档使用。

## 项目简介

home-leis-guide 是一个以 HTML 文件为主要内容的仓库，旨在整理和保存若干独立页面。每个页面均为自包含的 HTML 文档，可离线浏览或部署至任意静态托管服务。

## 目录说明

```
home-leis-guide/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的资源文件（如 CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/`：各 HTML 页面按需存放，文件名即页面标识。
- `assets/`：资源文件按页面或类型组织，便于引用和维护。

## 页面归档说明

- 每个页面均为独立 HTML 文件，不依赖外部数据库或后端服务。
- 页面内容基于公开资料整理，仅供存档和参考。
- 仓库不跟踪页面访问量，不收集用户信息。

## 维护说明

- 本仓库不定期更新，新增或修改页面时会在 commit 信息中简要说明。
- 欢迎通过 Issue 或 Pull Request 提交页面建议或修正。
- 请勿在页面中添加商业推广、诱导点击或违法内容。

## 许可

本仓库内容仅用于存档和学习交流。如需引用或转载，请保留原始来源信息。
